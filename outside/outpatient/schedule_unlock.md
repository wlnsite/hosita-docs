# 释放已锁定号源


- **接口说明：** 未缴费，释放已锁定号源
- **接口地址：** /api/outpatient/schedule_lock
- **请求方式：** GET/POST
- **请求参数：**
    | 参数名称 | 参数类型 | 出现要求 | 描述 |
    | -------- | -------- | -------- | ---- |
    | pre_id | string | R | 锁号时的预约单号 |
    | see_no | string | O | 锁号时的就诊序号 |

- **输出参数：**
    | 参数名称 | 参数类型 | 出现要求 | 描述 |
    | -------- | -------- | -------- | ---- |
    | code | string | R | 执行状态：1-成功，其它表示异常 |
    | message | string | R | 消息：错误消息或成功提示 |
    | data | object | R | 输出数据：释放号源结果 |
    | - result | bool | R | 是否成功，true或false |

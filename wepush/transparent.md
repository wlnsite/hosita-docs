# 通道透传接口

- **接口说明：** 消息通道原生接口透传通道
- **接口地址：** /push/transparent/{channel_id}
- **请求方式：** POST
- **请求参数：** 直接透传业务参数，请参考推送通道的官方文档

- **输出参数：**
    | 参数名称 | 参数类型 | 出现要求 | 描述 |
    | -------- | -------- | -------- | ---- |
    | code | string | R | 执行状态：0-成功，其它表示异常 |
    | success | bool   | R | 执行状态：业务是否成功      |
    | message | string | R | 消息：错误消息或成功提示 |
    | data | string | R | 通道返回的原始消息 |
